# Diagrams

This folder contains diagrams used across multiple books or concepts.

## Diagram Types

You can store:
- PNG or JPG diagrams
- SVG architecture diagrams
- Excalidraw exports
- Mermaid diagrams (if embedded in markdown, place them in the `.md` file, not here)

## Naming Convention

Use clear, descriptive filenames:

- bookname-chapterNN-topic.png
- concept-topic.svg
- generic-diagram-title.png

Examples:
- ddia-ch01-storage-engine.svg
- caching-flow.png
- event-sourcing-overview.png

## Guidelines

- Keep files lightweight.
- Always reference the diagram from the chapter or shared markdown file.
- If replacing a diagram, keep the filename the same unless necessary.
