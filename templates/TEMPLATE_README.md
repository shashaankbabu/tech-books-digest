# Templates Guide

This folder contains reusable templates for adding new books, chapters, and progress tracking to the repository.  
Contributors should refer to these templates before creating or editing content.

---

## chapter-template.md

Use this file when creating a new chapter inside a book folder.

Example location:
books/<book-name>/01-chapter-title.md

This template includes sections for:
- Key ideas
- Summary
- Deep insights
- Examples or diagrams
- Interesting points
- Real-world applications
- Questions or doubts

Each chapter file should follow this structure to keep the repository consistent.

---

## book-summary-template.md

Use this file to create a complete book summary after finishing a book.

Example location:
books/<book-name>/summary.md

This template includes:
- Overall summary
- Key takeaways
- Core concepts
- Practical applications
- Open questions
- Additional resources

This helps maintain a uniform format for book-level insights.

---

## progress-template.md

Use this file to track reading progress for a specific book.

Example location:
books/<book-name>/progress.md

The table helps track:
- Which chapters are completed
- Who completed them
- When they were completed
- Any notes or observations

---

## How to Use These Templates

1. Copy the relevant template file.
2. Rename it appropriately based on the book or chapter.
3. Place it inside the corresponding book folder.
4. Fill in each section with your notes or summaries.
5. Submit your changes through a Pull Request.

Following these templates ensures consistency and makes the repository easy to navigate.
